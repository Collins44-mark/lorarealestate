How to manage your listings

  1. Locations – Add locations first (e.g. "Oyster Bay", "Dar es Salaam"). Each property must belong to a location.
  2. Properties – Add or edit properties. Set title, price, type (sale/rent), and details. Upload main image and gallery images. Use Featured for homepage highlights. Use Published to show on the site.
  3. Inquiries – View and manage contact form submissions.
  4. Bookings – View and manage viewing requests (Pending, Confirmed, Cancelled).

For media uploads, ensure Cloudinary is configured in your environment.
