How to manage your listings

  1. Locations – Add locations first (e.g. "Oyster Bay", "Dar es Salaam"). Each property must belong to a location.
  2. Properties – Add or edit properties. Set title, price, type (sale/rent), details. Upload main image and gallery. Use Featured for homepage; Published to show on site.
  3. Bookings – View and manage viewing requests. Update status: Pending → Confirmed or Cancelled.
  4. Inquiries – View contact form submissions from the website.

For media uploads, ensure Cloudinary is configured in your environment.
